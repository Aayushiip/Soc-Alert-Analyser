# SOC-Alert-Analyzer

## Project Purpose
Beginner project to simulate SOC alert monitoring and compliance tracking.

## How to Run
1. Place CSV logs in `alerts/`
2. Run `scripts/parse_alerts.py` to count failed login attempts
3. Run `scripts/compliance_tracker.py` to check compliance task status

## Outputs
- Scripts print reports to console
- Optionally, outputs can be saved to `outputs/` folder
